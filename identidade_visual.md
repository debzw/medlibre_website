# Identidade Visual - MedLibre

MedLibre é uma plataforma freemium de preparação para residência médica que prioriza a fluidez, clareza e foco do estudante.

## 1. Nome da Marca
- **Nome:** medlibre
- **Estilo:** Todo em minúsculas, transmitindo modernidade e acessibilidade.

## 2. Logotipo
O logotipo é minimalista e combina conceitos de medicina ("L" de Libre/Libertas) com liberdade.

- **Logotipo Principal (Integrated):** `public/logo_withname.svg` (Light mode) / `public/logo_withname_white.svg` (Dark mode)
- **Ícone (Icon only):** `public/logo.svg`
- **Cor Primária:** `#EDB92E` (Sunflower Gold)
- **Conceito:** "Connected Line" (ML Crown).

## 4. Tipografia
O projeto utiliza fontes modernas e impactantes.

- **Headings (Títulos):** `Archivo Black` (Para H1 a H6)
- **Body (Corpo):** `Lexend Deca` (Para textos longos e interface)
- **Fallback:** `-apple-system, BlinkMacSystemFont, Segoe UI, Roboto, sans-serif`

## 5. Padrões de UI
- **Raio de Borda (Radius):** `0.75rem` (12px)
- **Sombra (Elevated Card):** `0 1px 3px rgba(0, 0, 0, 0.04), 0 6px 16px rgba(0, 0, 0, 0.04)`
- **Efeito Glass:** `bg-background/80 backdrop-blur-lg border-border/50`
- **Animações:** Transições suaves de `0.2s` a `0.4s` (`fade-in`, `slide-up`).
- **Cards:** Bordas suaves, fundo semi-transparente no dark mode.

## 6. Posicionamento Estratégico (O Jeito MedLibre)
A marca se posiciona como a alternativa inteligente aos cursinhos tradicionais.

- **Foco:** Aprendizado baseado em evidências (Active Recall + Spaced Repetition).
- **Tom de Voz:** Sofisticado, provocativo e focado em resultados reais.
- **O Inimigo:** O "Complexo Industrial das Videoaulas" e o estudo passivo.
- **Diferencial:** Algoritmos que direcionam o estudo para o erro e a revisão constante do que é mais necessário para cada aluno.
